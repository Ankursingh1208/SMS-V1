BINDING Syntax: 
	() : Even Binding syntax is used to pass the data to HTML to TypeScript.
	[] : Property Binding syntax is used to pass the data from TypeScript to HTML.
	[()] :This is Banana InBox syntax which is used for Two Way Data Binding. 


-> In order to use Two Way Data Binding we have to use a Built-in module knowns as "FormsModule"
-> Two Way Data Binding is important feature for Angular and very popular.
-> All form submission is typically done using Two Way Data Binding.

-> 	[(ngModel)] = "username" where
		ngModel :is an Built-in attribute given by Angular.
		username : is field defined in the ts file.
